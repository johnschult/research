What's happening under the hood

This page calls https://api.anthropic.com/v1/messages directly from your browser using the Fetch API. There is no server, no proxy, no backend of any kind.


Anthropic added CORS support in August 2024. To opt in, every request must include the header anthropic-dangerous-direct-browser-access: true — the name is intentional: it forces developers to acknowledge the security trade-off.
